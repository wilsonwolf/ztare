# Seed Specs

Seed specs are strategic inputs for supervisor-governed programs.

Layout:
- `active/`: seeds currently eligible for genesis-opened work
- `deferred/`: seeds intentionally postponed
- `legacy/`: superseded seed specs retained for provenance

These files define program intent. Debate, implementation, and verifier artifacts belong elsewhere:
- debates: `research_areas/debates/`
- prose/code drafts: `research_areas/drafts/`
- deterministic specs: `research_areas/specs/`
