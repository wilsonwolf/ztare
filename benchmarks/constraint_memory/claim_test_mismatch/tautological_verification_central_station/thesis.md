### ARITHMETIC TRANSPARENCY & SYMBOLIC MAPPING

1.  **Host-Ambassador Program Unit Economics (per 8-person cohort):**

    *   $CS\_GP_{program} = TotalProgramRevenue \times CS\_RevenueShare = \$480 \times 0.40 = \$192$
    *   $HostRevenue_{program} = TotalProgramRevenue \times (1 - CS\_RevenueShare) = \$480 \times 0.60 = \$288$
    *   $HostDirectCosts_{program} = HostDirectCostPerParticipant \times CohortSize = \$10 \times 8 = \$80$
    *   $HostNetProfit_{program} = HostRevenue_{program} - HostDirectCosts_{program} = \$288 - \$80 = \$208$
    *   **This confirms an attractive profit for hosts and a healthy gross profit for CS per program.**

2.  **Breakeven Micro-Geography Unit Economics (Monthly):**
    *   $MicroGeoFixedCosts_{monthly} = (HostRecruitmentCostPerHost \times TargetHostsPerMicroGeo) / 12 + (HostMgmtCostPerHostPerMonth \times TargetHostsPerMicroGeo) + LocalizedMarketingOverheadPerMonth$

    *   $MicroGeoFixedCosts_{monthly} = (\$200 \times 5) / 12 + (\$50 \times 5) + \$300 = \$83.33 + \$250 + \$300 = \$633.33$
    *   $ProgramsForBreakeven_{monthly} = MicroGeoFixedCosts_{monthly} / CS\_GP_{program} = \$633.33 / \$192 \approx 3.3$ programs
    *   $ParticipantsForBreakeven_{monthly} = ProgramsForBreakeven_{monthly} \times CohortSize = 3.3 \times 8 \approx 27$ participants
    *   $ActiveFullMembersForBreakeven_{monthly} = ParticipantsForBreakeven_{monthly} / AvgProgramParticipationPerMemberPerMonth = 27 / 0.33 \approx 82$ active full members
    *   **This defines a specific, falsifiable target density for a new micro-geography.**
